# Birch Learning Academy

Premium, nature-inspired website for Birch Learning Academy — a boutique early childhood education center in Johnson County, Kansas.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — scroll animations
- **React Hook Form + Zod** — form validation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
